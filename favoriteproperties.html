Introduction.

The purpose of this assignment, and website, is to show 5 CSS properties and how they benefit and help create web pages.

The resize Property

The "resize" property lets us resize and reposition the height and width of the div element. This is a very handy property because it allows us to resize boxes without making the context harder to read. *See example down below.

The padding Property

The "padding" property is also a very important property because it creates space around an element within a designated border. This is helpful for buttons and other things that need space to make it look nice.

This is an example of creating a defined space within a box using padding

The font Property

The "font" property allows us to change the style, variant, weight, size, and family. This allows us to give more depth to our text, and allows us to change the style and look of the text in only moments, and give more meaning to certain words and show importance on our site.

The animation Property

The "animation" property is essentially the shorthand property for the eight properties that animation has. Many different CSS properties has shorthands, but that doesn't necassiarly make them easier to use. The animation property is shorthand for name, duration, timing-function, delay, iteration-count, direction, fill-mode, and play-state.

The text-align Property

The "text-align" property isn't that interesting of a property, but it is definitly important. It allows us to align the text and give the site a huge appearance of order. It helps to justify within itself, and help show solid formatting.
